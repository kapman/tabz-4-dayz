# tabz-4-dayz
A chrome extension to manage your tabs with keyboard shortcuts!

## Features 
- Switch to left/right tab using `ctrl + j` & `ctrl + l`
- Toggle back & forth between last visited tabs using `cmd/alt + j` & `cmd/alt + shift + j`
